# Codex Skill Creator

This skill is a merged Codex-oriented skill creator package. It combines:

- The Codex-provided `skill-creator` skill from this local Codex installation.
- The Anthropic `skill-creator` skill from `https://github.com/anthropics/claude-plugins-official/tree/main/plugins/skill-creator/skills/skill-creator`.

The merged version keeps the richer evaluation and iteration assets from the Anthropic skill while adapting the workflow, metadata, scripts, and terminology for Codex.
